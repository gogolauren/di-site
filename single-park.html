Parking lot Navigation Software Design

Team Work (team leader: framework developer and UI designer) in "Fundamentals of Computer Science" class

Parking Lot Navigation is designed to help users find available parking lots. The app can search and recommend nearby parking lots depending on users' current location or their destination; then the app will map the most flexible route. Through positioning and searching, the map dynamically displays the location of park and the number of free spaces.


Real-time route display leads to frequent screen flashes and slows down the efficiency. To solve this problem, I reduced system's automatic refresh frequency, making it refresh only after each click; moreover, to address users’ concerns about the distribution of available parks, I mapped the actual location of parks and dynamically displayed the number of free spaces by incorporating Google map and studying Google's temporal and spatial behaviors.
